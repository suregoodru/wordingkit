# WordingKit Changelog

## [Initial Release] - {PR_MERGE_DATE}

### Added

- Open-source project documentation, license, contribution guide, security
  policy, issue templates, pull request template, and GitHub Actions CI.
- Rewrite selected text from any macOS app using AI
- Support for Ollama, OpenAI, Anthropic, and Groq providers
- Built-in Russian editing modes for grammar fixes, rewriting, clarity,
  shortening, expansion, friendlier tone, formal tone, natural wording, emoji,
  social posts, selling copy, and work chat
- Custom editing modes with provider, model, prompt, temperature, and token
  limit settings
- Manual or last-used sorting for editing modes
- Paste rewritten text back to active app with Enter
- API keys stored in Raycast Preferences
